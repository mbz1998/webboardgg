# Building a forum with Laravel
การสร้างเว็บบอร์ดด้วย Laravel 5.5

## แผนในการพัฒนา
* User Authentication - สมัครสมาชิก, เข้าสู่ระบบ, ออกจากระบบ และกู้คืนรหัสผ่าน;
* User Profile - แก้ไขข้อมูลส่วนตัว อัพรูปโปรไฟล์ และเปลี่ยนรหัสผ่าน;
